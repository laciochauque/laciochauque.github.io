Sobre mim

Sou um profissional de tecnologias de informação com uma sólida formação acadêmica e em activo desde 2022 na área. Ao longo dos anos, adquiri experiências valiosas em diversas facetas da informática, desde a configuração e manutenção de servidores e serviços em rede, programação de aplicações web até a resolução eficiente de problemas em sistemas operacionais Windows e Linux. Sou um entusiata da tecnologia desde criança, e desenvolvi uma enorme curiosidade nesta área e gostaria de puder partilhar minhas experiências e conhecimentos já adquiridos com as novas gerações afim de contribuir na elevação do meu país no ramo tecnológico.

Competências

Programação e Design Web

Como Programador e design web, trago uma sólida experiência no desenvolvimento de soluções web de ponta a ponta, desde a concepção até a implementação. Ao longo dos anos, aprimorei minhas habilidades em linguagens como HTML, CSS, JavaScript e PHP e frameworks como VueJs, Laravel e Codeigniter. Sou apaixonado por criar experiências digitais envolventes e funcionais, priorizando sempre a usabilidade e o desempenho. Além disso, valorizo a colaboração em equipe e estou sempre buscando aprender e me adaptar às novas tecnologias e melhores práticas da indústria. Estou empolgado com a oportunidade de contribuir em projetos inovadores e ajudar a impulsionar o sucesso digital.

Administração de Redes Informáticas

Tenho também formação sólida em administração de redes informáticas com foco em ecossistemas baseados em Linux, e trago uma vasta experiência em gerenciamento e otimização de infraestrutura de rede para garantir operações eficientes e seguras. Ao longo dos anos, tornei-me experiente em configuração, monitoramento e manutenção de servidores Linux, bem como em implementação de soluções de segurança, como firewalls com iptables e VPNs. Além disso, possuo habilidades sólidas em automação de tarefas por meio de ferramentas como Ansible e shell scripting. Estou sempre entusiasmado com a possibilidade de contribuir para o sucesso das pequenas e médias empresas desempenhando esta função.

Assistência Administrativa

Por ser técnico superior em informática, tenho alto domínio em Microsoft Office e Google Workspace, trago uma sólida habilidade em utilizar essas plataformas para aumentar a eficiência e a produtividade no ambiente de trabalho. Tenho habilidades para produzir e organizar documentos e impressões em série utilizando Word, criar planilhas e dashboards recorrendo Excel e VBA, produzir apresentações de alta qualidade no PowerPoint, gerir emails e calendários no Outlook até criar pequenas bases de dados utilizando Access. Além disso, estou familiarizado com as ferramentas colaborativas do Google Workspace, como o Google Docs, Sheets, Slides e forms, e sei como utilizá-las para facilitar a colaboração em equipe e o compartilhamento de informações de forma eficaz. Com um forte foco na organização e na atenção aos detalhes, estou preparado para apoiar as operações administrativas numa empresa de qualquer escalão garantindo que os processos sejam executados de maneira eficiente e profissional.

Rota de Aprendizagem
MS Office (2018)

MS Office é uma colleção de aplicativos desktop de escritórios distribuídos pela Microsoft para utilizadores de computadores pessoais.

Google Workspace (2019)

Google Workspace é uma colleção de aplicativos web de escritórios distribuídos pela Google na nuvem.

Linux Basics (2019)

Linux é um sistema operativo amplamente utilizado em servidores, é de código livre e é a base da maioria dos sistema utilizados hoje em dia como o caso de Android.

HTML5 (2020)

HTML5 é uma linguagem de marcação para a estrutura e apresentação de conteúdos na World Wide Web

CSS3(2020)

Cascading Style Sheets é uma linguagem de estilização para documentos escritos numa linguagem de marcação, como HTML ou XML

JavaScript(2020)

O JavaScript é uma linguagem de programação usada para criar conteúdo dinâmico, controlar multimídia, animar imagens e muito mais.

VueJs(2020)

Vue.js é um framework progressivo de JavaScript para construção de interfaces de usuário em páginas web, focado na reatividade e na simplicidade, facilitando a criação de aplicações web interativas e dinâmicas.

PHP(2021)

PHP: Hypertext Preprocessor é uma linguagem de programação amplamente utilizada para pré-processamento server-side do conteúdo HTML por apresentar numa página web.

MySQL(2023)

MySQL é um sistema de gerenciamento de banco de dados relacional (SGBD) de código aberto amplamente utilizado em aplicações web

SASS(2021)

Syntactically Awesome Style Sheets SASS é uma extensão do CSS. Ela permite escrever estilos de forma mais eficiente e modular, com recursos como variáveis, aninhamento, mixins e herança

Bootstrap(2021)

Bootstrap é um framework client-side que fornece uma coleção de ferramentas e componentes pré-estilizados para agilizar o desenvolvimento de páginas web.

Codeigniter(2021)

CodeIgniter é um framework de PHP, que segue o padrão Model-View-Controller. Ele oferece um conjunto de bibliotecas e ferramentas para facilitar a criação de aplicações web, enfatizando a simplicidade, o desempenho e a facilidade de aprendizado.

Tailwinds(2021)

Tailwind CSS é um framework de desenvolvimento client-side que utiliza uma abordagem utility-first para a criação de interfaces de usuário.

AlpineJs(2021)

Alpine.js é uma biblioteca JavaScript minimalista que permite adicionar interatividade e comportamento dinâmico às páginas web diretamente no HTML, sem a necessidade de escrever muito código JavaScript.

Laravel(2022)

Laravel é o mais robusto framework de PHP, que segue o padrão Model-View-Controller. Ele é ideal para aplicações web complexas.

Laravel Livewire(2022)

Laravel Livewire é uma biblioteca para o framework PHP Laravel que permite criar interfaces de usuário interativas de forma simples e sem a necessidade de escrever JavaScript.
